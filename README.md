# 📱 Quiz App

A lightweight, mobile-first quiz application built with vanilla web technologies and modern JavaScript.

## Overview

This project is a responsive quiz app designed specifically for mobile screen sizes. It runs entirely in the browser with no frameworks or dependencies — just clean, modern HTML, CSS, and JavaScript.

## Tech Stack

| Technology | Version / Standard             |
| ---------- | ------------------------------ |
| HTML       | HTML5                          |
| CSS        | CSS3                           |
| JavaScript | ES2025 (ECMAScript 2026 draft) |

## Features

- Mobile-first responsive design
- Vanilla JavaScript — no frameworks or libraries
- Lightweight and fast — runs entirely in the browser

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/quiz-app.git
   ```
2. Open `index.html` in your browser — no build step required.
